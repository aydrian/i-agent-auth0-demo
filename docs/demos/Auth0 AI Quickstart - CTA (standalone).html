Three pillars · One control plane

Auth0 for AI Agents.

01
Identity
The agent knows who the user is. Auth0 session resolved at request time, threaded into the system prompt with the user's name and linked accounts.
02
Token Vault
The agent calls third-party APIs on the user's behalf. Refresh tokens live in Auth0; your app never holds them. Scopes pinned per integration.
03
CIBA
Human-in-the-loop for high-stakes actions. The agent pushes consent to the user's phone with a binding message it composed itself.
The agent owned the decision; you owned the consent.
